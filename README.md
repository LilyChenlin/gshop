# gshop外卖app

> A Vue.js project

## 开发资源
- 使用阿里巴巴矢量库,将需要的图标加入库生成项目图标的Font Class地址
- 项目目录static/css/reset.css添加样式重置文件

## Vue插件及第三方库
- 使用vue-router开发单页应用
- 使用 axios 与后端进行数据交互
- 使用 vuex 管理应用组件状态
- 使用 better-scroll/vue-scroller 实现页面滑动效果
- 使用 mint-ui 组件库构建界面
- 使用 vue-lazyload 实现图片惰加载
- 使用 mockjs 模拟后台数据接口
